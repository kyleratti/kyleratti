Why I Code

When I was 12, a seemingly insignificant event that forever changed my life: I went to a summer camp and played a Star Wars video game.

This led to a chance meeting with a game modification developer named @RoboPhred who took the time to teach me how to program my own modifications. He has since become a life-long friend and mentor!

As it turns out, I like knowing how stuff works and I really like building stuff that others can use. From video mirror bots and mirror directory listing bots for Reddit communities, to auto-updating slideshow websites and HTTP file relay servers for my sister's work, I think it's the coolest thing in the world to be able to build something, watch it come to life, and see it used by others!
