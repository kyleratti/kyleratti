Why I Code

When I was 12, a seemingly insignificant event changed my life forever: I went to a summer camp and played a Star Wars video game.

My new-found obsession with the game eventually led to a chance virtual meeting with @RoboPhred, the developer of a popular modification. He took the time to show me how to program my own modifications for the game and create web pages, and has since become a life-long mentor and friend!

As it turns out, I like knowing how stuff works and I really like building stuff that others can use. From video mirror bots and mirror directory listing bots for Reddit communities, to auto-updating slideshow websites and HTTP file relay servers for my sister's work, I think it's the coolest thing in the world to be able to build something, watch it come to life, and see it used by others!
